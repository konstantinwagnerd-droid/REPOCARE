# PDL-Dashboard-Walkthrough

**4 Seiten. Deine Wochenroutine mit CareAI als Pflegedienst-Leitung.**

---

## Seite 1 — Der Montag: Woche planen

### 08:00 — Morgen-Check (7 Minuten)

Öffne das **PDL-Dashboard** (Homepage nach Login). Du siehst:

- **Status-Kachel Wohnbereiche:** grün / gelb / rot auf Blick.
- **Offene Aufgaben:** Quittungen fehlend, Maßnahmen überfällig.
- **Anomalien über Nacht:** KI-Flags wie „ungewöhnlicher Gewichtsverlust Bewohner 12".

**Workflow:**
1. Wohnbereich auf Rot? → Klick, Detail ansehen, delegieren.
2. Anomalie? → Klick → Detail-Chart → Rücksprache mit zuständiger Pflegekraft.
3. Offene Aufgaben priorisieren via Drag&Drop.

### 09:00 — Dienstplan-Review (10 Minuten)

Dashboard → „Dienstplan". Solver zeigt grüne/rote Schichten:
- Grüne: ausreichend besetzt.
- Rote: Lücken mit Vorschlägen („Maria Hofer nächste Rotation, hat 32h offen").

**Aktion:** Lücken schließen via Klick. Benachrichtigung geht automatisch raus.

---

## Seite 2 — Dienstag bis Donnerstag: Qualität

### 10:00 — SIS-Übersicht

Dashboard → „SIS-Qualität". Bewohner-Matrix: Wer hat diese Woche einen aktuellen SIS-Eintrag, wer nicht?

**Regel:** Jede:r Bewohner:in alle 14 Tage mindestens 1 aktualisierter SIS.
- Rot = überfällig → Pflegekraft zuordnen.
- Gelb = fällig diese Woche → im Schichtbuch vermerken.

### 11:00 — Wunddokumentations-Review

Dashboard → „Wunden". Alle offenen Wunden mit Heilungsverlauf-Chart. Bei stagnierenden oder verschlechternden:
- Fotoserie ansehen.
- Arzt-Kontakt initiieren via „Arzt benachrichtigen"-Button (E-Mail mit Dokumentation).

### 14:00 — MD-Prüfungs-Bereitschaft

Dashboard → „MD-Readiness". Score-Balken pro Bewohner (0–100%).

Klick auf Bewohner mit < 85% → Liste fehlender Elemente:
- „Pflegeanamnese älter als 6 Monate"
- „Risikoassessment Sturz fehlt"
- „Biographie unvollständig"

→ Pflegekraft direkt beauftragen per In-App-Nachricht.

---

## Seite 3 — Freitag: Abschluss & Berichte

### 08:00 — Wochenbericht generieren

Dashboard → „Wochenbericht". Auto-generiertes PDF mit:
- Auslastung
- Kritische Ereignisse (Stürze, Hospitalisierungen)
- Personalstunden (Soll/Ist)
- Anomalien der Woche
- QM-Kennzahlen

Du prüfst, ergänzt Kommentare. Ein Klick → an Geschäftsführung / Träger.

### 10:00 — Team-Feedback sammeln

Dashboard → „Team-Puls". Anonymisierte Stimmungsbild-Umfrage (5 Fragen, freiwillig).

Trend über Wochen sichtbar. Abfallender Trend im Wohnbereich 3 → Einzelgespräch mit Team-Lead.

### 12:00 — Woche nachmelden

Dashboard → „Abrechnungsexport". Alle Pflegedienstleistungen für Kassa-Abrechnung. CSV-Export, Import in Buchhaltung.

---

## Seite 4 — Monatsroutine

### Monatsende (60 Minuten)

1. **Dashboard → „Monats-QM-Report"** — auto-generiert, prüfen und versenden.
2. **Medikations-Audit:** welche Abweichungen wurden erfasst? Trend?
3. **Sturz-Statistik:** Bewohner mit > 2 Stürzen/Monat → erneute Risikoassessment.
4. **Personal-Kennzahlen:** Überstunden, Krankenstände, Rotation.

### Monatsgespräche

Dashboard → „Bewohner-Review" → Filter „Monatsgespräche fällig". Liste für Angehörigen-Kontakt inkl. Gesprächsleitfaden.

---

## Power-User-Tipps

- **Kurztasten:** `Strg+K` = Universalsuche (Bewohner, Mitarbeiter, Einträge).
- **Eigene Dashboards:** bis zu 3 personalisierbar. Kacheln per Drag anordnen.
- **Automatische Reports:** Einstellungen → „Planer". Jeden Freitag 17:00 Wochenbericht per Mail — ohne Klick.
- **Angehörigen-Anfrage-Workflow:** einmal einrichten, spart pro Woche 2h Telefonate.
