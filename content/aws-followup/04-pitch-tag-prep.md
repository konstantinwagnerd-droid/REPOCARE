# Mail 04 — Pitch-Tag-Prep

## Kontext
Wenn die aws zum Pitch-Tag einlaedt (positiv — Shortlist erreicht).
Mail bestaetigt Teilnahme + Logistik, enthaelt interne Checkliste.

## Betreff
Bestaetigung Teilnahme aws Preseed Pitch-Tag [Datum]

## Preview-Text
Bestaetigung + kurze Rueckfragen zur Logistik.

## Body

Sehr geehrtes aws-Team,

herzlichen Dank fuer die Einladung zum Pitch-Tag am [Datum] in [Ort].
Ich bestaetige hiermit meine Teilnahme.

**Organisatorische Rueckfragen:**

1. **Dauer-Format:** Ist die vorgesehene Pitch-Dauer [x] Minuten plus
   [y] Minuten Q&A? Habe ich das richtig verstanden?

2. **Technik:** Stellen Sie Beamer/Praesentations-Equipment? Format
   16:9, PowerPoint/PDF oder Keynote? HDMI oder USB-C-Adapter
   vorhanden?

3. **Jury-Zusammensetzung:** Darf ich erfahren, wer die Jury-Mitglieder
   sind, damit ich passende Backgrounds vorbereiten kann?

4. **Reihenfolge:** Gibt es eine feste Slot-Zuteilung oder wird das am
   Tag vor Ort bekanntgegeben?

5. **Materialien vorab:** Soll ich die Pitch-Deck-PDF vorab einreichen
   oder erst am Tag?

Ich plane, [x Min] vor Start vor Ort zu sein, um die Technik zu testen.

Fuer den Fall, dass ich im Anschluss an den Pitch-Tag noch fuer kurze
Gespraeche zur Verfuegung stehen soll: Gerne, ich kann bis [Uhrzeit]
bleiben.

Mit freundlichen Gruessen

Konstantin Wagner
Gruender, CareAI
konstantin@careai.at | careai.at

## INTERNE CHECKLIST — Nicht mitsenden!

### 7 Tage vorher
- [ ] Pitch-Deck final gereviewt (mind. 1 Advisor-Feedback)
- [ ] Speaker-Script geuebt (3x komplett durch)
- [ ] 20 Q&A-Fragen durchgegangen
- [ ] Pitch mit Stoppuhr: Zeitlimit einhalten
- [ ] 5 Peers/Advisors-Proberuns absolviert

### 3 Tage vorher
- [ ] Technische Backup-Strategie (Laptop, USB-Stick mit PDF, Cloud-Link)
- [ ] Outfit getestet (business-casual, CareAI-Farben dezent)
- [ ] Visitenkarten gedruckt (100 Stueck)
- [ ] One-Pager-Handout gedruckt (30 Stueck)
- [ ] Demo-Video Offline-Version auf Laptop

### 1 Tag vorher
- [ ] Anreise geklaert (oeffentlich bevorzugt)
- [ ] Hotel gebucht (falls noetig, nahe Venue)
- [ ] Alkoholfrei, vor 23:00 ins Bett
- [ ] Wasser + Snacks fuer den Tag
- [ ] Aufgeladene Ersatz-Batterien / Powerbank

### Am Tag
- [ ] 45 Min vorher am Venue
- [ ] Technik-Test (Adapter, Lautstaerke, Demo-Video)
- [ ] 3 tiefe Atemzuege vor Pitch-Start
- [ ] Ruhe bewahren, Augenkontakt, laecheln
- [ ] Bei Q&A: 2 Sek denken, dann antworten
- [ ] Nach Pitch: allen Jury-Mitgliedern Hand schuetteln + Visitenkarte

### Nach dem Tag
- [ ] Innerhalb 24h: Thank-You-Mail an Jury (siehe 05-post-pitch-followup.md)
- [ ] 48h: Reflexion — was lief gut / schlecht? Dokumentieren
- [ ] Warten auf offizielle Entscheidung (kein Draengeln)
