# Storyboard — 8-Minuten Heimleitungs-Produkt-Demo

## KAPITEL 1 · 0:00–0:45 · BEGRÜSSUNG

```
+------------------------------------------+
|                                          |
|   [Presenter, Halbportrait, sitzt]      |
|   Hintergrund: Pflegeheim-Büro unscharf  |
|   Natürliches Licht von links            |
|                                          |
+------------------------------------------+
```
Presenter direkt in Kamera. Lavalier-Mikrofon. Kein Studio-Glanz — echt.
